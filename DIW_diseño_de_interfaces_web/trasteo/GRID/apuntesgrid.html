Apuntes Extensivos de CSS Grid

4. Uso de nombres de áreas

Podemos definir nombres a las áreas de un grid y organizarlas de forma más clara.

Ejemplo:

grid-template-areas:
"header header header"
"sidebar content content"
"footer footer footer";

Ejemplo: Grid con Áreas Nombradas

Header
Sidebar
Content
Footer

5. Uso de auto-fill y minmax

La función auto-fill junto con minmax() permite crear columnas dinámicas que se ajustan al tamaño disponible.

grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));
